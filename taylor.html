Método Serie de Taylor

Inicio Serie de Taylor Método de Bisección

Explicación

La serie de Taylor es una representación de una función como una suma infinita de términos calculados a partir de las derivadas de la función en un punto.

La fórmula para la serie de Taylor centrada en a es:

f(x) ≈ Σn=0∞ [f⁽ⁿ⁾(a)/n!] (x - a)ⁿ

Resolución paso a paso

Ejemplo: Aproximar e^x en x=1 usando la serie de Taylor centrada en a=0 hasta el término 4.

  1. Calcular derivadas: para e^x todas son e^x, evaluadas en 0 = 1
  2. Calcular términos: 1 + x + x²/2! + x³/3! + x⁴/4!
  3. Sustituir x=1 y sumar para aproximar

Código Python

import math
import matplotlib.pyplot as plt
import numpy as np

def taylor_exp(x, a=0, n_terms=5):
    suma = 0
    for n in range(n_terms):
        suma += (math.exp(a) / math.factorial(n)) * (x - a)**n
    return suma

x = np.linspace(-2, 2, 100)
y_exacta = np.exp(x)
y_taylor = [taylor_exp(xi) for xi in x]

plt.plot(x, y_exacta, label="e^x")
plt.plot(x, y_taylor, label="Serie de Taylor (5 términos)")
plt.legend()
plt.grid(True)
plt.title("Aproximación con Serie de Taylor")
plt.savefig("img/taylor_plot.png")
plt.show()
