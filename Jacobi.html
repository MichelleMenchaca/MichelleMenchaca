Método de Jacobi

Inicio Método de Jacobi

Explicación

El método de Jacobi es un método iterativo para resolver sistemas de ecuaciones lineales que descompone la matriz en su diagonal y el resto para calcular nuevas aproximaciones.

Resolución paso a paso

  1. Se despeja cada variable en función de las demás usando la diagonal de la matriz.
  2. Se elige una solución inicial.
  3. Se iteran las fórmulas hasta que la diferencia entre iteraciones sea menor a una tolerancia dada.

Código Python

import numpy as np

def jacobi(A, b, x0=None, tol=1e-8, max_iter=100):
    n = len(b)
    x = np.zeros_like(b) if x0 is None else x0.copy()
    D = np.diag(A)
    R = A - np.diagflat(D)

    for _ in range(max_iter):
        x_new = (b - np.dot(R, x)) / D
        if np.linalg.norm(x_new - x, ord=np.inf) < tol:
            return x_new
        x = x_new
    raise Exception("No converge")

# Ejemplo
A = np.array([[4, -1, 0, 0],
              [-1, 4, -1, 0],
              [0, -1, 4, -1],
              [0, 0, -1, 3]])
b = np.array([15, 10, 10, 10])
sol = jacobi(A, b)
print(f"Solución: {sol}")
