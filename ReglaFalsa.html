Método de la Regla Falsa

Inicio Serie de Taylor Método de Bisección Regla Falsa

Explicación

El método de la Regla Falsa es un método iterativo para encontrar raíces de una función, que mejora el método de bisección usando una línea secante para aproximar la raíz.

Resolución paso a paso

  1. Elegir intervalos [a, b] donde f(a) y f(b) tengan signos opuestos.
  2. Calcular el punto de intersección c = b - f(b)(b - a)/(f(b) - f(a)).
  3. Evaluar f(c). Si es cero o menor que la tolerancia, c es la raíz.
  4. Actualizar el intervalo sustituyendo a o b según el signo de f(c).
  5. Repetir hasta alcanzar la precisión deseada.

Código Python

def regla_falsa(f, a, b, tol=1e-6, max_iter=100):
    for _ in range(max_iter):
        fa, fb = f(a), f(b)
        c = b - fb * (b - a) / (fb - fa)
        fc = f(c)
        if abs(fc) < tol:
            return c
        if fa * fc < 0:
            b = c
        else:
            a = c
    return c

# Ejemplo
f = lambda x: x**3 - x - 2
raiz = regla_falsa(f, 1, 2)
print(f"Raíz aproximada: {raiz}")
