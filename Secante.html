Método de la Secante

Inicio Serie de Taylor Método de Bisección Método de la Secante

Explicación

El método de la secante es un método iterativo para encontrar raíces que utiliza dos aproximaciones iniciales y la pendiente de la línea secante para calcular nuevas aproximaciones.

Resolución paso a paso

  1. Elegir dos valores iniciales x₀ y x₁.
  2. Calcular la siguiente aproximación con la fórmula: xₙ₊₁ = xₙ - f(xₙ)*(xₙ - xₙ₋₁)/(f(xₙ) - f(xₙ₋₁)).
  3. Repetir hasta alcanzar la tolerancia deseada.

Código Python

def secante(f, x0, x1, tol=1e-6, max_iter=100):
    for _ in range(max_iter):
        fx0 = f(x0)
        fx1 = f(x1)
        if fx1 - fx0 == 0:
            raise ZeroDivisionError("División por cero en secante.")
        x2 = x1 - fx1 * (x1 - x0) / (fx1 - fx0)
        if abs(x2 - x1) < tol:
            return x2
        x0, x1 = x1, x2
    return x2

# Ejemplo
f = lambda x: x**3 - x - 2
raiz = secante(f, 1, 2)
print(f"Raíz aproximada: {raiz}")
