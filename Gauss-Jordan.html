Método de Gauss-Jordan

Inicio Serie de Taylor Método de Bisección Gauss-Jordan

Explicación

El método de Gauss-Jordan extiende la eliminación gaussiana para reducir la matriz aumentada a la forma reducida por filas (matriz identidad), facilitando la solución directa del sistema.

Resolución paso a paso

  1. Formar la matriz aumentada [A|b].
  2. Usar operaciones fila para obtener la matriz identidad en la parte izquierda.
  3. La solución está entonces en la parte derecha de la matriz aumentada.

Código Python

import numpy as np

def gauss_jordan(A, b):
    n = len(b)
    M = np.hstack([A.astype(float), b.reshape(-1,1)])
    for i in range(n):
        # Normalizar fila i
        M[i] = M[i] / M[i,i]
        # Eliminar otras filas
        for j in range(n):
            if j != i:
                M[j] = M[j] - M[j,i] * M[i]
    return M[:, -1]

# Ejemplo
A = np.array([[2,1,-1],[-3,-1,2],[-2,1,2]])
b = np.array([8,-11,-3])
sol = gauss_jordan(A, b)
print(f"Solución: {sol}")
