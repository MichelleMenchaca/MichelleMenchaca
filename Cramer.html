Regla de Cramer

Inicio Serie de Taylor Método de Bisección Regla de Cramer

Explicación

La regla de Cramer utiliza determinantes para resolver sistemas de ecuaciones lineales con el mismo número de ecuaciones y incógnitas.

Resolución paso a paso

  1. Calcular el determinante de la matriz de coeficientes det(A).
  2. Para cada incógnita, reemplazar la columna correspondiente por el vector de términos independientes y calcular su determinante.
  3. La solución para cada incógnita es el cociente entre estos determinantes: x_i = det(A_i)/det(A).

Código Python

import numpy as np

def regla_cramer(A, b):
    det_A = np.linalg.det(A)
    if det_A == 0:
        raise ValueError("El sistema no tiene solución única")
    n = len(b)
    x = np.zeros(n)
    for i in range(n):
        A_i = A.copy()
        A_i[:, i] = b
        x[i] = np.linalg.det(A_i) / det_A
    return x

# Ejemplo
A = np.array([[2, -1, 3], [1, 0, -2], [3, 1, 1]])
b = np.array([5, -1, 4])
sol = regla_cramer(A, b)
print(f"Solución: {sol}")
