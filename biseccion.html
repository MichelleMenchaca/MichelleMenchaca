Método de Bisección

Inicio Serie de Taylor Método de Bisección

Explicación

El método de bisección es una técnica numérica para encontrar raíces de una función continua f(x) en un intervalo [a, b] donde la función cambia de signo (es decir, f(a)*f(b) < 0).

Consiste en dividir repetidamente el intervalo a la mitad y seleccionar el subintervalo donde la función cambia de signo, garantizando la convergencia hacia la raíz.

Resolución paso a paso

Supongamos que queremos encontrar la raíz de f(x) = x^3 - x - 2 en el intervalo [1, 2].

  1. Calcular el punto medio: m = (a + b)/2
  2. Evaluar f(m)
  3. Determinar en qué subintervalo la función cambia de signo y actualizar a o b
  4. Repetir hasta alcanzar la tolerancia deseada

Código Python

def f(x):
    return x**3 - x - 2

def biseccion(a, b, tol):
    if f(a)*f(b) >= 0:
        print("No se puede aplicar el método en este intervalo.")
        return None

    m = (a + b)/2
    while abs(f(m)) > tol:
        if f(a)*f(m) < 0:
            b = m
        else:
            a = m
        m = (a + b)/2
    return m

raiz = biseccion(1, 2, 0.0001)
print(f"Raíz aproximada: {raiz}")
