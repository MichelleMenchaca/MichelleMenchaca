Método de Newton-Raphson

Inicio Serie de Taylor Método de Bisección Newton-Raphson

Explicación

El método de Newton-Raphson es un método iterativo para aproximar raíces de funciones utilizando la derivada de la función para mejorar la aproximación en cada paso.

Resolución paso a paso

  1. Escoger una aproximación inicial x₀.
  2. Calcular la siguiente aproximación: xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ).
  3. Repetir hasta que la diferencia entre aproximaciones sea menor que una tolerancia.

Código Python

def newton_raphson(f, df, x0, tol=1e-6, max_iter=100):
    x = x0
    for _ in range(max_iter):
        fx = f(x)
        dfx = df(x)
        if dfx == 0:
            raise ZeroDivisionError("Derivada cero, no se puede continuar.")
        x_new = x - fx / dfx
        if abs(x_new - x) < tol:
            return x_new
        x = x_new
    return x

# Ejemplo
f = lambda x: x**3 - x - 2
df = lambda x: 3*x**2 - 1
raiz = newton_raphson(f, df, 1.5)
print(f"Raíz aproximada: {raiz}")
