Aproximación por mínimos cuadrados

Inicio Mínimos cuadrados

Explicación

La aproximación por mínimos cuadrados busca encontrar la curva que mejor se ajusta a un conjunto de datos minimizando la suma de los cuadrados de las diferencias entre los valores observados y los valores estimados.

Resolución paso a paso

  1. Formar el sistema normal AᵀA x = Aᵀb.
  2. Resolver el sistema para obtener los coeficientes del polinomio de ajuste.
  3. Evaluar o usar el polinomio para predecir valores.

Código Python

import numpy as np

def minimos_cuadrados(x_points, y_points, grado):
    A = np.vander(x_points, grado + 1)
    coef = np.linalg.lstsq(A, y_points, rcond=None)[0]
    return coef

# Ejemplo
x_points = np.array([0, 1, 2, 3])
y_points = np.array([1, 2, 1.3, 3.75])
coeficientes = minimos_cuadrados(x_points, y_points, grado=2)
print(f"Coeficientes del polinomio: {coeficientes}")
