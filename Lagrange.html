Interpolación de Lagrange

Inicio Interpolación Lagrange

Explicación

El polinomio de interpolación de Lagrange se construye como una suma ponderada de polinomios base que son 1 en un punto dado y 0 en los demás.

Resolución paso a paso

  1. Definir los polinomios base Li(x).
  2. Multiplicar cada Li(x) por el valor yi correspondiente.
  3. Sumar todos los términos para obtener el polinomio interpolador.

Código Python

import numpy as np

def lagrange_interpolation(x_points, y_points, x):
    total = 0
    n = len(x_points)
    for i in range(n):
        term = y_points[i]
        for j in range(n):
            if j != i:
                term *= (x - x_points[j]) / (x_points[i] - x_points[j])
        total += term
    return total

# Ejemplo
x_points = np.array([0, 1, 2])
y_points = np.array([1, 3, 2])
valor = lagrange_interpolation(x_points, y_points, 1.5)
print(f"Valor interpolado en 1.5: {valor}")
