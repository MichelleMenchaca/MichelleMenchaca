Método de la Inversa

Inicio Serie de Taylor Método de Bisección Método de la Inversa

Explicación

El método de la inversa resuelve sistemas de ecuaciones lineales calculando la inversa de la matriz de coeficientes y multiplicándola por el vector de términos independientes.

Resolución paso a paso

  1. Calcular la matriz inversa A⁻¹ de la matriz de coeficientes A.
  2. Multiplicar la inversa por el vector b: x = A⁻¹b.

Código Python

import numpy as np

def metodo_inversa(A, b):
    A_inv = np.linalg.inv(A)
    x = A_inv @ b
    return x

# Ejemplo
A = np.array([[3,1],[1,2]])
b = np.array([9,8])
sol = metodo_inversa(A, b)
print(f"Solución: {sol}")
