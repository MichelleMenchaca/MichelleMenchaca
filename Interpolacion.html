Interpolación resolviendo Ax=b

Inicio Interpolación Ax=b

Explicación

La interpolación puede resolverse formulando un sistema lineal donde la matriz contiene potencias de los puntos dados y se resuelve para los coeficientes del polinomio.

Resolución paso a paso

  1. Construir la matriz A donde cada fila es [1, x_i, x_i², ..., x_iⁿ].
  2. Construir el vector b con los valores y_i.
  3. Resolver Ax = b para obtener los coeficientes.

Código Python

import numpy as np

def interpolacion_axb(x_points, y_points):
    n = len(x_points)
    A = np.vander(x_points, increasing=True)
    b = np.array(y_points)
    coef = np.linalg.solve(A, b)
    return coef

# Ejemplo
x_points = np.array([1, 2, 3])
y_points = np.array([2, 3, 5])
coeficientes = interpolacion_axb(x_points, y_points)
print(f"Coeficientes del polinomio: {coeficientes}")
